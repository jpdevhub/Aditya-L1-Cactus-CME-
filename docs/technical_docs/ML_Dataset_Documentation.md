## Feature Categories

### CACTUS CME Features
- `cme_velocity`: CME propagation velocity (km/s)
- `cme_width`: Angular width (degrees)
- `pa`: Position angle (degrees)
- `halo_class`: Halo classification (1-4)
- `velocity_class`: Velocity classification (Slow/Medium/Fast/Very Fast)
- `earth_directed`: Binary flag for Earth-directed CMEs

### APEX SWISS Solar Wind Features
- `proton_density`: Proton number density (cm⁻³)
- `proton_bulk_speed`: Solar wind speed (km/s)
- `proton_temperature`: Proton temperature (K)
- `alpha_density`: Alpha particle density (cm⁻³)
- `alpha_proton_ratio`: Alpha to proton ratio

### 12-Hour Time Strip Features
For each parameter, statistical measures (mean, std, max, min, trend) are calculated
for the following time windows:
- `pre_cme_12h`: 12 hours before CME
- `post_cme_12h`: 12 hours after CME
- `pre_arrival_12h`: 12 hours before expected arrival
- `post_arrival_12h`: 12 hours after expected arrival

### Derived Features
- `cme_kinetic_energy`: Proportional to v²
- `cme_momentum`: Velocity × angular width
- `cme_hour`: Hour of CME detection
- `cme_day_of_year`: Day of year
- `cme_month`: Month of detection
